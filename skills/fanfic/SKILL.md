# 同人创作与版权检测（fanfic）

帮作者写同人，**同时守住版权红线**。同人是热门赛道，但踩线轻则锁文、重则被诉。

## ⚠️ 免责
本技能提供创作实务建议，**非法律意见**。商用变现前请取得 IP 方书面授权或咨询专业人士。

## 何时用
- 想写某个 IP 的同人（影视/动漫/小说/游戏/综艺）
- 问"这个同人能不能签约赚钱 / 在哪个平台发"
- 给已有同人做**版权风险检测**
- 需要"安全改写"方案规避侵权

## 两种模式

### A. 同人版权检测（合规校验）
对一个同人创意/稿子做风险体检（调 `references/fanfic-guide.md`）：
```
【IP 版权状态】公版(>50年) / 在版受保护 → 风险等级
【目标平台政策】该平台能否同人签约/变现（晋江✅唯一合法 / 番茄UC知乎❌ / 海外较宽松）
【借用角色数】__ 个（建议≤3 且大改）
【主线原创度】100%原创 / 含原著剧情 → 风险
【原文引用】（晋江规则：≤全文1/10 且单篇≤3000字）
【OOC/审核风险】是否恶意抹黑、三观、未成年等红线
【结论】可商用 / 仅免费 / 高风险勿碰
【整改建议】怎么改到安全线内
```

### B. 同人创作（安全方案）
帮用户写同人，但内置合规：
1. 先确认 IP 和目标平台 → 判断能不能变现
2. **优先推荐公版 IP**（西游/三国/红楼/封神/聊斋）做安全二创
3. 受保护 IP：建议"借名字不借主线"——角色≤3、大改人设、100% 原创主线
4. 按网文手法写（钩子/爽点照常，调其他技能）
5. 写完自动跑一遍 A 的合规检测

## 安全度铁律
- **公版 IP 最安全**：作者去世超 50 年的作品可自由商用二创。
- **受保护 IP 不授权 = 不能收费**：免费发也可能被下架/起诉。
- **晋江是国内唯一同人合法变现区**，且有引用比例硬规则。
- **海外同人较宽松**（海外华人写影视/网文同人是蓝海），但跨境商用仍建议授权/公版。
