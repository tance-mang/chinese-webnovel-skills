# 金手指设计（goldfinger）

金手指是男频网文的发动机。烂金手指让主角变工具人，好金手指让"努力和选择"被放大。

## 何时用
- 设计系统 / 签到 / 空间 / 模拟器 / 词条 / 复制吞噬 / 重生先知 / 召唤 等外挂
- 给主角设计变强机制
- 用户的金手指"太万能没张力"或"获取太随意"

## 工作流

### 第 1 步：选流派
根据频道、题材、平台从流派库选型 → 调 `references/goldfinger-types.md`
（系统流/签到流/空间流/模拟器流/词条流/复制吞噬/重生先知/召唤流/赘婿扮猪/无限流）

### 第 2 步：四要素设计
为选中的金手指设计：
1. **获取门槛**：怎么得到的？要独特、绑定主角遭遇/性格，不能路边捡。
2. **成长曲线**：分阶段台阶，每阶解锁新玩法，列出 3–5 个升级节点。
3. **使用代价**：冷却 / 反噬 / 资源消耗 / 道德取舍——制造张力，避免无敌即无聊。
4. **主线绑定**：来源/进化挂钩主线大谜团（谁给的？为什么是我？），留后期戏份。

### 第 3 步：上线与首秀
- 上线时机：番茄第 1 章 / 起点≤第 3 章（调 `platform-profiles.md`）
- 设计"小试牛刀"：上线后 1–2 章内用它解决一个具体困境，给第一个爽点

### 第 4 步：防烂尾设计
- 给金手指设"成长上限的悬念"，避免主角无敌后无戏可写
- 规划后期金手指的进化/觉醒大节点

## 输出格式
```
【金手指名称】
【流派】
【获取方式】（含独特性说明）
【核心机制】一句话
【成长台阶】阶段1→2→3→4→5（各解锁什么）
【使用代价】
【主线绑定】来源谜团 + 后期进化方向
【上线时机 + 首秀困境】
```

## 避坑
- ❌ 万能无代价 ❌ 获取随意 ❌ 主角全靠外挂无能动性 ❌ 与世界观脱节
- ✅ 金手指是放大器，放大主角的选择和努力

## 频道提示
女频较少用硬金手指，多用"身份/重生/读心/空间"软外挂，重点在关系和信息差而非数值，见 `references/channel-guide.md`。
