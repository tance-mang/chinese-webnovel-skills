# 男频女频分流指南（Channel Guide）

> 男频和女频是两套不同的"爽"逻辑。写之前先定频道，节奏、爽点、人物、情感配比全不同。
> 本库供所有写作技能在判断"该往哪个方向写"时调用。

---

## 一、核心差异

| 维度 | 男频 | 女频 |
|---|---|---|
| **底层爽点** | 实力升级、资源扩张、打脸对手、地位攀升（我变强、掌控全局） | 情感拉扯 + 独一份偏爱、女主自我成长、仇人报应（被偏爱 + 自我圆满） |
| **代入方式** | 成为强者（向上） | 被珍视/掌控全局（向内 or 俯视） |
| **剧情主线** | 世界线推进，从弱小→登顶，世界观越写越大 | 人际关系线推进，CP 磨合、误会反转、双向奔赴 |
| **开篇逻辑** | 主角受辱 → 金手指落地 → 当场打脸 | 女主身陷困境（被背叛/穿书/被害）→ 男主登场/开启自救线 |
| **金手指** | 硬外挂（系统/功法/空间） | 软外挂（身份/重生/读心/空间） |
| **冲突** | 实力冲突、势力争斗 | 情感冲突、身份冲突、家族博弈 |
| **反派** | 挡路的强者 | 抢爱的、害我的、看不起我的 |
| **爽点节奏** | 打脸→升级→更大的脸 | 误会→反转→双向奔赴/打脸白莲 |
| **结局诉求** | 站上巅峰、一览众山小 | 圆满的爱 + 体面的赢 |

---

## 二、男频主要类型

- **玄幻/修真**：升级流，宗门、功法、天才打脸。番茄/起点主力。
- **都市**：战神归来、赘婿、医武、扮猪吃老虎。
- **系统流**：签到、任务、数值，番茄爆款集中地。
- **历史/种田**：策略、经营、争霸。
- **科幻/末世**：空间、进化、求生。

**男频铁律**：主角要有**主观能动性**，金手指是放大器不是替身；爽点要**升级**不能原地踏步。

---

## 三、女频主要类型

- **古言/宅斗宫斗**：身份、算计、打脸白莲与恶毒女配（作者素材库主场景）。
- **现言/豪门**：虐恋、重生复仇、打脸渣男渣女。
- **穿书/快穿**：穿进剧情改命，反套路。
- **纯爱/甜宠**：情绪流，双向奔赴。

**女频铁律**：情感线是主菜，实力线是配菜；"被偏爱"和"掌控感"是核心爽点；打脸要**体面、解气、不掉价**。

---

## 四、女频爽点的特殊性（结合作者素材）

作者素材库中的开篇范例（穿越女斗世家、白绫挂断、替身逆位、挡刀反转）体现女频典型爽感：
- **弱势开局 + 不认命**：身份弱（穿越女/替身/庶女）但姿态强（决绝反抗）。
- **关系反转**：替身→正主、网恋对象→恶霸本人，靠**关系的意外**制造钩子。
- **情绪解气**：女频的"打脸"更重情绪解气（让瞧不起我的人后悔），而非单纯实力碾压。
