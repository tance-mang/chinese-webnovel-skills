# 网络热梗 · 实时查 + 不突兀（slang）

热梗能让现代向网文有网感，但**用错地方就立刻廉价又出戏**。本技能负责"搜到当下的梗"+"用得不突兀"。

## 何时用
- 想给现代都市/校园/职场/现实向小说加点网感
- 想知道当下抖音/小红书(RedNote)/微博/B站(TikTok/Douyin) 流行什么梗
- 问某个梗"现在还火不火 / 会不会过气了"
- 担心"加梗突兀、显廉价"

## ⚠️ 时效性
热梗每月在变，**优先用联网搜索（WebSearch）拉当前的梗**；无联网时基于通用知识给判断并**明确标注"可能已过期，建议联网核对"**。

## 三种用法

### A. 查当前热梗（联网）
用 WebSearch 检索（示例）：
- "抖音/小红书/微博 最新 热梗 流行语 <当前年月>"
- "<某个梗> 还火吗 出处 用法"
聚合多条、去重，输出：梗 + 含义 + 出处 + 适用场景 + **是否还在火（标时间）**。

### B. 怎么用不突兀（重点）
给定场景/题材/人物，按 6 条铁律判断（调 `references/slang-guide.md`）：
1. 看题材（古风/玄幻/正剧 → 不用现代梗）
2. 贴人物身份（年轻有网感才说）
3. 适量（一篇 1–3 个）
4. 避开快过气的
5. 放进对话/心理，别硬塞旁白
6. 服务情绪/人物，不为玩而玩

输出：能不能用 / 用哪个 / 放哪句 / 怎么融，给"示例改写"。

### C. 体检（查突兀/过气/语域不符）
扫稿子里的梗，标出：
- 题材/语域不符的（古风里的现代梗 → 删）
- 可能已过气的（→ 提示核对或换）
- 硬塞旁白、堆太多的（→ 减量、移进对话）

## 原则
- **宁可不用，不要硬蹭**：拿不准就不用，比用错强。
- **越"一句话场景梗"越短命，越"概念词"越长青**。
- 梗是调味，不是主菜；少而准。
