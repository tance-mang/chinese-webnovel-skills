# 网文起名（title）

书名决定点击率。本技能专治"AI 味/大众味"——AI 爱起抽象文艺名，网文要的是**关键词 + 爽点 + 钩子**。

## 何时用
- 起新书名 / 改名 / 要一批候选
- 嫌名字"太普通、太 AI、太文艺、没记忆点"
- 给已有书名做点击率优化

## 工作流

### 第 1 步：明确信息
确认 频道 + 平台 + 题材 + 核心卖点（金手指/爽点）。缺就问，或从已有设定取。
（番茄/七猫爱长名带卖点；起点偏精炼——调 `references/platform-profiles.md`）

### 第 2 步：起 6-8 个候选（分风格）
从起名套路各取几种（调 `references/title-library.md`），覆盖不同风格供挑：
- 关键词+夸张型、第一人称口语型、身份反差型、开局体、系统题材型、反套路调侃型…
每个名字必须满足**反 AI 三要素**：
1. 带**题材关键词**（系统/读心/侧写/赘婿/重生…，让算法识别）
2. 带**爽点或钩子**（无敌/打脸/倒计时/十年悬案/跪求…）
3. 用**具体意象**，不用抽象形容词（恐怖屋＞惊魂；倒计时＞宿命）

### 第 3 步：标注 + 自检
每个候选标：风格 + 卖点 + 适配平台。然后过"起名自检清单"（`title-library.md`），剔除：
- 抽象文艺名（《浮世清欢》类）
- 烂大街通用名（《重生之路》类）
- 没题材关键词的名

### 第 4 步：推荐 + 说明
挑 2-3 个最优，说明为什么点击率高；可给"安全款"和"大胆款"各一。

## 输出格式
```
【题材/频道/平台】
【推荐书名】Top 2-3（标注：为什么抓人）
【全部候选】
  1.《___》关键词型 — 卖点：___ — 适配：番茄
  2.《___》第一人称型 — …
  …
【避坑提醒】哪些常见起法会显 AI/大众，已规避
```

## 原则
- **宁可土爽，不要文艺空**：网文书名是广告，不是诗。
- 没题材关键词的名一律重起（算法识别不出 = 推不出去）。
- 抽象形容词换成具体意象。
- 热门关键词会变，需要时用 `trends` 联网查当下平台热词。

## 反例对照（教学用）
- ❌《暗夜追凶》→ ✅《读心刑警：我能看见凶手的犯罪倒计时》
- ❌《重生之路》→ ✅《重生后我把渣男全家送进了局子》
- ❌《修仙日记》→ ✅《我的灵田能种出神仙》
