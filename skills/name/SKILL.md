# 角色起名（name）

专治角色名"太大众、太 AI"——AI 一起就是叶辰、林轩、苏念、沈清欢，姓全挤在叶/萧/林/楚。本技能起**贴时代、贴身份、有记忆点、不撞大众款**的名。

## 何时用
- 给主角/女主/反派/配角/长辈起名或改名
- 嫌角色名烂大街、太 AI、没记忆点
- "我不会起名字，帮我起一批"
- 给宗门/势力/地名命名

## 工作流

### 第 1 步：明确信息
确认：**时代/世界观**（古风/现代/西幻/修仙）+ **角色身份性格** + **性别** + 起几个。缺就问。
（时代决定风格——古人不能叫现代名，调 `references/name-library.md`）

### 第 2 步：按 5 步法起名
1. 定时代风格 → 2. 选姓（**避开 叶/萧/林/楚/苏 等烂大街网文姓**，从姓氏池挑有质感的）→ 3. 取体现性格/命运的字（不堆生僻字）→ 4. 配音韵（好念、声调有起伏）→ 5. 查重查歧义。

### 第 3 步：成批给候选 + 说明
每个角色给 4-6 个候选，**每个标注**：为什么贴这个角色、记忆点在哪。
- 反派优先给"文雅冷静"款（比脸谱化的"黑X王"瘆人）。
- 一组配角名互相**拉开区分度**（别都两字、别都一个调）。

### 第 4 步：自检
过"起名自检清单"（`name-library.md`）：避烂大街姓、不撞知名角色、无生僻字、贴时代、配角能区分。

## 输出格式
```
【时代/世界观】
【角色：身份+性格】
  候选：名1（记忆点/为什么贴）、名2、名3、名4…
【宗门/势力/地名】（如需）
【避坑说明】规避了哪些烂大街起法
```

## 原则
- **避开网文流水线姓**：叶/萧/林/楚/苏（女主）/慕容——一看就 AI。
- **利落 > 软糯**：女主用"野/照/辞/棠"比"清欢/婉儿"更出挑、更不撞。
- **反派要瘆**：文雅冷静的反派名最高级。
- **别堆生僻字**：读者打不出、记不住的名是大忌。
- **贴时代**：长辈用年代名才真实，古人别叫现代名。

## 反例对照
- ❌ 男主 叶辰/林轩 → ✅ 陆沉 / 霍珩 / 纪川（避网文姓，沉稳有记忆点）
- ❌ 女主 苏念/沈清欢 → ✅ 温野 / 裴知秋 / 江照（利落有反差）
- ❌ 反派 血煞魔君 → ✅ 谢明烛 / 宋邃（文雅冷静更瘆人）
