# 拆书对标爆款（deconstruct）

爆款不是抄来的，是**拆出结构、复用到自己题材**。本技能把一本火书"拆零件"，提炼可复用的写作模板。

## ⚠️ 原创底线
**学结构、学套路、学节奏——不抄原文、不照搬剧情、不复制人物**。
套路（如打脸、扮猪吃老虎）是公共写作技法，可自由借鉴；具体文字、独创设定、剧情桥段受版权保护，不可复制。涉及同人/IP 见 `fanfic-guide.md`。

## 何时用
- "这本书为什么火，帮我拆一下"
- 对标某本爆款，想学它的写法
- 提炼爆文的开篇/爽点/金手指/节奏模板
- 想"照爆款结构写我自己的故事"

## 工作流

### 第 1 步：获取对标对象
用户提供书名/简介/章节文本，或描述这本书。
（要查它在平台的真实热度/同类爆款，用 `trends` 联网。已有拆解范例见 `references/case-studies.md`，可参照其拆解维度。）

### 第 2 步：六维拆解
按网文要素拆出**结构**（不复述原文）：
```
【对标书】+ 平台 + 频道 + 题材
【一句话爆点】它的核心卖点是什么
【开篇结构】用了哪种开篇钩（调 hook-library）+ 几句立危机
【金手指】什么流派、怎么上线、怎么成长（调 goldfinger-types）
【爽点节奏】爽点类型 + 密度 + 升级链（调 trope-library）
【人物模板】主角记忆点 + 反派强度 + 关系张力
【章节节奏】单章字数、章尾钩规律、付费节点设计
【成功要素】它做对了哪 3 件事
```

### 第 3 步：提炼可复用模板
把"它的结构"抽象成"你能用的模板"：
```
【可复用模板】
- 开篇公式：______（套你的题材）
- 金手指框架：______（换你的设定）
- 爽点链：______（换你的对手/场景）
- 节奏配方：______
【套用示范】用你的题材填一遍这个模板（原创内容）
【差异化提醒】别 1:1 复刻，至少换一个维度做出新鲜感
```

### 第 4 步：衔接创作
把模板喂给 `idea`/`outline`/`spark`，生成你**自己原创**的书。

## 原则
- 输出的是**抽象结构和方法**，不是原文段落。
- 永远提醒"换皮不够，要做差异化"（调 `trope-library.md` 反套路一节）。
- 多本同类爆款一起拆，能看出"这个题材的公共成功公式"。

## 下一步
模板提炼好 → `idea` 落选题、`outline` 排大纲、`spark` 直接孵化初稿。
