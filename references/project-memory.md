# 创作档案规范（Project Memory）

> 定义一本书的持久化记忆结构与各档案格式。供 `memory` 及所有写作技能读写。

---

## 一、档案目录约定
```
《书名》/
├── 00_设定.md        立项卡 + 金手指 + 世界观 + 势力库 + 战力体系
├── 01_人物.md        人设卡 + 角色当前状态 + 语言指纹
├── 02_大纲.md        卷纲 / 章纲
├── 03_伏笔追踪.md     埋点 → 回收 表
├── 04_灵感本.md       灵感速记
├── 05_进度.md        进度 / 字数 / 下一步
├── 06_投稿记录.md     投稿历史 + 拒因
├── 07_时间线.md       主时间线 + 知情人名单（长篇防失忆，见 skills/continuity）
└── 正文/第NNN章.md
```
> 短篇可只用 00/01/02；长篇务必维护 07_时间线 与 战力体系，否则写到上百章必崩（见 `continuity-check.md`）。

## 二、进度档（05_进度.md）格式
```
# 进度
- 当前章节：第 42 章
- 累计字数：约 11.8 万
- 当前卷：第二卷《XX》（共 5 卷）
- 上章结尾钩：主角踏入秘境，门在身后合拢
- 下一步：写第 43 章 —— 秘境第一关，初遇对手
- 最近更新：2026-06-05
```

## 三、伏笔追踪（03_伏笔追踪.md）格式
> 加**编号**便于跨章引用，加**重要度**（主线/支线/细节）便于排回收优先级——主线坑必收，细节坑可留白（见 `human-texture.md`）。

| 编号 | 伏笔 | 埋点章 | 计划回收章 | 重要度 | 状态 |
|---|---|---|---|---|---|
| F001 | 青铜钥匙 | 第3章 | 第50章左右 | 主线 | 待回收 |
| F002 | 师父的真实身份 | 第12章 | 第二卷末 | 主线 | 待回收 |
| F003 | 反派的弱点 | 第20章 | 第38章 | 支线 | ✅已回收(第38章) |

写到接近"计划回收章"还没回收的，`review`/`continuity` 会提醒。

## 四、投稿记录（06_投稿记录.md）格式 —— 记录"为什么没过"
> 目的：把每次失败变成可复用的经验，下次投稿前先读这张表，避免重犯。

```
# 投稿记录

## 第1次 · 番茄 · 男频系统流 · 2026-05-20
- 结果：❌ 未签约
- 编辑/系统反馈：开篇节奏慢，前三章没有明确爽点
- 我的判断：第1章铺垫太多世界观
- 改动：砍掉第1章前半，金手指提前到第800字
- 复盘标签：#开篇慢 #爽点晚

## 第2次 · 番茄 · 男频系统流(改版) · 2026-05-28
- 结果：✅ 已签约
- 反馈：开篇抓人，追读达标
- 复盘标签：#成功 #开篇前置爽点
```

**复盘标签**用于归纳高频拒因（如 `#开篇慢` `#人设崩` `#爽点晚` `#同质化`），投稿技能据此提示作者。

## 五、时间线（07_时间线.md）格式 —— 长篇防失忆
> 用 **Day / 第N月 / 第N年** 这类绝对锚点，别只记"几天后"。每章把"经过多久"落到这条线上。

```
# 主时间线
- Day1   开局，主角穿越/重生，绑定金手指
- Day3   第一次冲突，结识配角A
- Day10  入宗门（第5章）
- 第2月  闭关突破（第12章），设定出关=第3月
- 第3月  出关，赴秘境之约（第18章）
- 倒计时：中毒「七日内必解」——Day40 中毒，须 Day47 前解

# 知情人名单（谁知道主角的秘密）
- 真实身份(穿越者)：无人知 / 仅系统
- 真实修为：师父(第12章起)、女主(第20章起)
- 马甲「黑衣人」：暂无人识破
```

## 六、战力 / 力量体系（并入 00_设定.md）格式
> 谁强谁弱要有可对照的刻度，防越级打平、配角暴打 BOSS。

```
# 力量体系（境界从低到高）
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 大乘
- 跨大境界 = 碾压；同境界看功法/法宝/经验

# 战力对照（关键角色当前）
| 角色 | 当前境界 | 备注 |
|---|---|---|
| 主角 | 金丹初期 | 有金手指，可越级一档 |
| 反派A | 元婴中期 | 第三卷大敌 |
| 女主 | 筑基后期 | |
```

## 七、人物语言指纹（speech_profile，并入 01_人物.md）格式
> 防长篇"一人一腔最后变成所有人一个口气"。每个主要角色记一张，写对话时对照注入（见 `skills/dialogue`）。

```
## 张三 · 语言指纹
- 风格：粗豪直接，没耐心
- 口癖：「他娘的」「废什么话」
- 句长：偏短，爱用祈使句
- 常用词：干、揍、滚
- 不会说：文绉绉的成语、客套话

## 李四 · 语言指纹
- 风格：文雅克制，绵里藏针
- 口癖：「在下以为」「未必吧」
- 句长：偏长，爱反问
- 常用词：倒是、何妨、未尝
```

## 八、与 git 的关系
- 本档案是**纯文本**，天然适合 git 版本控制。
- 推荐在书文件夹 `git init`，定稿即提交，获得真正的历史/回滚/diff。
- 不用 git 也行：`memory` 技能提供带日期的轻量快照。
