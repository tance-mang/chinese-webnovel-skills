# 英文网文出海（english）

帮中国作者写**像当地作者**的英文网文，而不是"中文翻译腔"。核心：**中文卖剧情爽点，英文卖情绪钩子；英文读者要被直接扔进故事**。一切规则调 `references/english-webfic.md`。

## 🔛 总开关：Native English Mode（默认开启）
写英文时**默认**按欧美母语网文叙事，不是"中文思维→翻译成英文"，而是"英文读者本来就会这样表达"。这条改的是叙事思维，最重要：
- **优先动作、对话、冲突、悬念；降低解释、降低抒情、降低文学化描写。**
- **提高人物真实感和口语化表达；让读者沉浸在事件中，不旁观作者讲故事。**
- 严格执行 `english-webfic.md` 第〇节的**十条核心规则**（禁翻译腔/Show>Tell/句长70-20-10/禁文学堆砌/每200字推进/钩子优先/对话口语/母语短句/减副词/章尾留钩）。
- 结构按英文逻辑：**谜团 → 情绪 → 爽点**（先埋疑问再给爽）；爽感包装用**个人成长**（Survivor/Underdog/Outcast/Chosen One），不是开局身份碾压。

## 何时用
- 直接写英文小说（Wattpad / Royal Road / Webnovel / Radish / KDP）
- 把中文故事**改写**成英文原生（不是逐句翻译）
- 问"出海写英文怎么写 / 选哪个平台"
- 去掉英文里的翻译腔 / AI 腔

## 四种用法

### A. 写英文原生稿
按英文网文规则直接写英文（调 `english-webfic.md`）：
- **前 100 词有事发生**，第一句就是钩子（身份反转/危机/秘密/禁忌关系）。
- **短段落、短句**；对话**一句一行、换行**。
- **不解释、不堆形容词**，直接动作。
- 情绪用**身体反应**（`I couldn't breathe.` / `My hands shook.`），不喊抽象。
- 章标题 `Chapter X`。

### B. 中转英（改写成英文原生）
不是翻译，是**用英文重讲**：保留剧情/钩子/反转，按英文读者习惯重写节奏、段落、情绪表达；砍掉中文式铺垫和解释；去文化梗。

### C. 去英文 AI 腔/翻译腔
扫英文稿，标出并改：
- 太正式：`Therefore/However/Furthermore` → 口语或删。
- 太工整机械：`He smiled. She smiled.` → 打破对称。
- 过度文学：`Moonlight danced gracefully...` → 直白快。
- 长段落 → 拆短；对话挤一行 → 换行。

### D. 选平台
按频道/题材推荐（调 `english-webfic.md` 平台表）：
- 女性向言情 → **Wattpad**（最易上手）/ Radish
- 男频系统/升级/LitRPG → **Royal Road**
- 系统/重生/都市（像番茄）→ **Webnovel**
- 完结本卖电子书 → **KDP**

## 原则
- **像本地作者，不像翻译**：宁可简单直白，不要中式华丽。
- **情绪钩子 > 剧情解释**：把读者扔进情绪，别替他总结。
- 你的反转/钩子/快节奏是优势，适合 Wattpad 短篇 / Webnovel / 短视频小说流。

## 配合
英文也要去 AI 味 → `deslop`/`human` 的思路通用（但用 `english-webfic.md` 的英文 AI 破绽清单）；钩子结构 → `hook`。
