# 全品类题材库（Genre Library）

> 网文题材谱系。供 `idea` / `outline` / `spark` / `trends` 选型时调用。
> 用法：技能可带题材参数，如 `/webnovel:idea 犯罪心理探案`、`/webnovel:outline 系统都市`。
> 每个题材标注主力**频道**与适配**平台**（详见 `platform-profiles.md`）。

---

## 一、悬疑 · 探案 · 推理（含心理学向）

### 1. 本格推理
诡计作案、密室杀人、不在场证明、连环凶案，重线索/物证/逻辑解谜。
- 细分：密室推理、暴风雪山庄、连环预告杀人、暗号密码破案

### 2. 社会派探案
案件映射现实矛盾（原生家庭/贫富/职场黑暗），案件只是载体。
- 细分：扫黑刑侦、失踪人口、陈年积案、家暴衍生命案

### 3. 刑侦纪实（公安体制）
一线刑警/法医/痕检/禁毒/网安/重案组，依托真实办案流程。
- 细分：法医探案、卧底刑侦、跨省追凶、缉毒大案

### 4. 心理向悬疑（犯罪心理学/变态心理学）
1. **犯罪心理学**：连环杀手心理侧写、反社会人格、偏执型罪犯、动机溯源
2. **变态心理**：人格分裂、病态占有欲、创伤后应激作案、童年阴影连环行凶、控制型罪犯
3. **心理博弈**：警匪心理对战、囚徒博弈、催眠犯罪、精神病伪装脱罪
- 细分：心理侧写师、犯罪心理顾问、微表情破案、精神病院秘案

### 5. 悬疑小众
无限流悬疑、孤岛悬疑、民俗怪谈推理、阴间判官破案、穿越古代断案（古风刑探）

> 频道：男频为主（知乎盐选女频也吃悬疑脑洞）｜平台：知乎盐选短篇 / 番茄 / 起点

## 二、都市（男频主力 + 现实向）

### 男频都市
1. 都市神医：祖传医术、针灸、妙手医权贵
2. 都市战神/退役兵王：隐龙回归、落地都市复仇护亲
3. 赘婿逆袭：隐忍上门女婿、隐藏豪门、后期打脸势利亲戚
4. 都市鉴宝：古玩捡漏、赌石、古董修复、寻宝
5. 商业重生：重生 90/00 年代、创业炒股、地产互联网、白手起家首富
6. 异能都市：透视、控火控水、自愈、动植物沟通
7. 市井生活：摆摊、开小店、乡村进城创业、小人物逆袭

### 现实都市（知乎盐选爆款）
职场内卷、房产纠纷、原生家庭、婆媳矛盾、婚恋背叛、反 PUA、律师行业文
> 想写温暖治愈、有现实意义但不狗血不鸡汤的 → 用 `warmth` 技能 + `warm-realism.md`。

> 频道：男频/现实向通吃｜平台：番茄/起点（男频）、知乎盐选/UC（现实向）

## 三、系统流（网文最大类目，几十种系统）

> 系统 = 开局绑定，完成任务领奖励，是番茄/起点顶流模板（机制详见 `goldfinger-types.md`）。

1. **职业系统**：神医/法医/刑警/神厨/鉴宝/汽修/教师/律师 系统
2. **赚钱经商系统**：首富养成/地摊暴富/牧场庄园/连锁开店/股市神投
3. **修仙玄幻系统**：签到修仙/挂机修炼/打怪爆装备/宗门宗主
4. **生活休闲系统**：美食探店/宠物培育/绿植/钓鱼封神
5. **特殊脑洞系统**：反派作死/被吐槽变强/好人福报/捡垃圾暴富/情绪收割
6. **穿书系统**：炮灰保命/反派逆袭/男主替身/修正剧情

> 频道：男频核心（女频多用穿书系统）｜平台：番茄/起点

## 四、玄幻 / 仙侠 / 奇幻

1. **古典修仙**：炼气→筑基→金丹→元婴 升级流、宗门争斗、灵根异变、秘境寻宝
2. **洪荒神话**：盘古、巫妖、封神、上古神兽、人族崛起
3. **西幻奇幻**：魔法师、骑士、龙族、地下城冒险、精灵矮人异世界
4. **修真种田**：灵田种仙草、养灵宠、小宗门安稳发育

> 频道：男频核心｜平台：起点/番茄/Webnovel（出海最吃香）

## 五、女频专项（晋江 + 番茄女频）

1. **现言**：霸总甜宠、先婚后爱、追妻火葬场、破镜重圆、娱乐圈顶流
2. **古言**：宅斗宫斗、侯门庶女、王妃逆袭、帝王独宠
3. **穿书**：穿成恶毒女配、炮灰自救、原书女主对线
4. **年代文**：六零/七零/八零、下乡知青、养家致富、重生囤货
5. **纯爱（晋江）**：强强、师徒、破镜、末世基建、古代官场耽美

> 频道：女频｜平台：晋江（精品）、番茄（下沉甜宠/虐恋）

## 六、历史 / 古代 / 架空（重点补全）

> 你特别问到的"中国古代、架空历史"，这里单列补全。

1. **历史穿越**：唐宋明清穿越做官、造兵器搞工业、辅佐帝王、改写历史走向
2. **架空历史**：虚构王朝/架空背景的权谋争霸、开国争霸、登基流（不绑定真实朝代，规避考据与审核）
3. **古代种田经营**：穿成古代小农，开荒种地、养殖、开店发家、古代美食/商业
4. **古代宅斗宫斗**（女频）：庶女逆袭、后宫争宠、世家联姻博弈
5. **古风权谋/朝堂**：文官武将、党争、变法、边关战事
6. **古代探案**（古风刑探）：仵作验尸、县令断案、江湖悬案
7. **古典仙侠/武侠**：江湖门派、快意恩仇、侠之大者

> 频道：男频（争霸/权谋/历史）、女频（宅斗/宫斗/古言）｜平台：起点/晋江/番茄

## 七、灵异 / 惊悚 / 民俗

1. **民俗灵异**：东北出马、阴阳先生、风水先生、坟地怪事、乡村诡异
2. **阴间题材**：阴间公务员、摆渡亡魂、地府开店、收鬼契约
3. **恐怖无限流**：惊悚副本、密室闯关、怪谈规则、诡异旅馆、规则怪谈

> 频道：男频为主｜平台：番茄/UC（民俗）、起点（无限流）

---

## 八、年代 / 乡土 / 种田

1. **古代种田**：穿成古代小农，开荒、养殖、开店发家（见第六类）
2. **现代乡村**：返乡创业、果园、养殖、带全村致富
3. **重生年代**：70/80 年代囤粮囤货、避灾、改善家人命运

> 频道：男频（乡土战神/种田）、女频（年代养家）｜平台：UC/番茄

## 九、小众 / 增量题材

1. **末世求生**：丧尸末世、天灾极寒、囤货避难、基地建设
2. **星际科幻**：星际战舰、挖矿、外星文明、星际帝国
